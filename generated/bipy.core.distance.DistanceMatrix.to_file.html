bipy.core.distance.DistanceMatrix.to_file¶

DistanceMatrix.to_file(out_f, delimiter='t')¶

Save the distance matrix to file in delimited text format.

Parameters:

out_f : file-like object

File-like object to write serialized data to. Must have a write method. It is the caller’s responsibility to close out_f when done (if necessary).

delimiter : str, optional

Delimiter used to separate elements in output format.
